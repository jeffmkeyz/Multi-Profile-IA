📌 Cómo funciona

Cada perfil funciona como un perfil de tu navegador. Adentro registras las cuentas de IA que usas ahí. Si le dices cuántos mensajes tiene cada una y cada cuántas horas se resetea, esta app calcula sola si ya puedes volver a usarla — y arriba siempre ves qué tienes disponible ahora destacando sus colores corporativos.

Todo vive en el servidor, no en este navegador: puedes abrirlo desde cualquier dispositivo con tu mismo Telegram.
